# rust-restful-msg-board

A simple rust restful message board. A simple practice for rust / angular / k8s / docker.


## Commands

### Run the dockers

```sh
docker-compose up -d
```

### Stop the dockers

```sh
docker-compose down
```

### list the image

```sh
docker list image
```

### list the containers

```sh
docker ps
```
